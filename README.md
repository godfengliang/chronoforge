# ChronoForge — Interactive Timeline Editor

Plan projects with drag-and-drop Gantt charts, right in your browser. No signup, no install.

**Live Demo:** [chronoforge-timeline.surge.sh](https://chronoforge-timeline.surge.sh/)

## Features

- **Drag & drop** — Move tasks by dragging, resize from edges
- **Dependencies** — Automatic dependency arrows between tasks
- **Progress tracking** — Set completion percentage per task
- **8 colors** — Color-coded task categories
- **Zoom** — Ctrl+scroll or buttons to zoom in/out
- **Today line** — Red indicator for current date
- **Weekend shading** — Visual weekends on the timeline
- **Fit all** — Auto-zoom to show all tasks
- **Task editor** — Double-click to edit name, dates, progress, color
- **Sidebar** — Task list with color indicators
- **PNG export** — Download the timeline as an image
- **Responsive** — Works on different screen sizes

## Tech

- Canvas 2D API for timeline rendering
- DDA-style day calculation for infinite scroll
- Drag-and-drop with resize handles
- Single HTML file, zero dependencies
